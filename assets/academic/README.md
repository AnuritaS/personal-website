# Academic background photos

Drop college / campus photos here as:

    college-1.jpg
    college-2.jpg
    college-3.jpg
    ...
    college-12.jpg

These tile together as a full-page mosaic background behind the
Academic page's text (same idea as assets/travel/, just used as a
background layer instead of an inline gallery). Any aspect ratio
works — each tile is cropped with object-fit: cover.

If a file is missing, that tile quietly falls back to a soft
placeholder pattern instead of a broken image icon.
