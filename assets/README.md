# Assets folder

Drop your headshot/portrait here as:

    profile.jpg

Recommended: a vertical crop (portrait orientation), face and shoulders
clearly framed, similar to a standard professional headshot. The hero
section uses a 4:5 aspect ratio crop (slightly taller than wide) at
max-width 280px, so a reasonably high-res image (at least 800x1000px)
will look sharp.

If no profile.jpg is present, the homepage automatically falls back to
the "AS" initials avatar — no broken image icon, no manual toggle needed.
